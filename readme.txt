Adds the ability to flag a product as discontinued to WooCommerce

== Description ==

This plugin adds the ability to mark a product in your WooCommerce inventory as discontinued.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Unzip the `.zip` file into the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
